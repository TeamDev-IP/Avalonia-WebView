# Avalonia Browser

A minimal browser application built with Avalonia UI and DotNetBrowser.

## Prerequisites

* .NET 6.0

## How to run

Execute the following command from the solution directory:

```
dotnet run --project DotNetBrowser.AvaloniaUi.Demo
```
